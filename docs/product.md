# Product

## プロダクト概要

**link-hub** は、アカウントごとにブックマークを管理するシンプルな Web アプリケーションです。

## 対象ユーザー

- URL を整理して管理したい個人ユーザー
- ブラウザに依存せず、どこからでもブックマークにアクセスしたいユーザー

## ゴール

- ログインしたユーザーが自分専用のブックマーク一覧を持てる
- ブックマークの登録・編集・削除をシンプルな UI で行える
- 他ユーザーのブックマークは参照・操作できない（完全なデータ分離）

## 成功指標

| 指標 | 内容 |
|------|------|
| ブックマーク登録 | ログインユーザーが URL・タイトル・メモを登録できる |
| ブックマーク一覧表示 | 自分のブックマークのみ一覧表示される |
| データ分離 | 他ユーザーのブックマークが参照・操作できない |
| 認証保護 | 未ログインユーザーがブックマーク画面にアクセスできない |

## 非ゴール（MVP スコープ外）

- ブックマークの共有・公開機能
- タグ・フォルダによる分類機能
- ブラウザ拡張連携
- インポート・エクスポート機能
