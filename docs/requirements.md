# Requirements

## 機能要件

### タグ管理

| # | 要件 |
|---|------|
| F-25 | ブックマーク登録・編集時にタグを複数付与できる（既存タグから選択 or 新規入力） |
| F-26 | タグはユーザー単位で管理される（他ユーザーのタグは参照・操作できない） |
| F-27 | 一覧画面でタグをクリックして絞り込みフィルタリングができる（複数選択可） |
| F-28 | タグが付いていないブックマークは「タグなし」フィルターで絞り込める |
| F-29 | ブックマーク一覧の各行からインラインでタグを設定・変更できる（編集画面に遷移不要） |
| F-30 | チェックボックスで複数選択したブックマークにタグを一括付与できる（既存タグは維持） |
| F-31 | タグ管理ページ（`/bookmarks/tags`）でタグの作成・削除・一覧表示ができる |

### 認証

| # | 要件 |
|---|------|
| F-01 | Clerk を使用してログイン・ログアウトができる |
| F-02 | 未ログインユーザーはブックマーク画面にアクセスできない（ログインページにリダイレクト） |
| F-03 | ログイン後、初回アクセス時に DB へユーザー情報を同期する |

### ブックマーク管理

| # | 要件 |
|---|------|
| F-10 | ログインユーザーは自分のブックマーク一覧を閲覧できる |
| F-11 | ブックマークを新規登録できる（URL・タイトル・メモ） |
| F-12 | ブックマークを編集できる（URL・タイトル・メモ） |
| F-13 | ブックマークを削除できる |
| F-14 | 他ユーザーのブックマークは参照・操作できない |

### 検索・表示改善

| # | 要件 |
|---|------|
| F-15 | キーワード検索ができる（title / url / memo に対する部分一致） |
| F-16 | URL 入力時に OGP を自動取得し、タイトル・OGP 画像 URL を補完できる |
| F-19 | ダークモードに対応する（OS 設定に従い初期化、ローカルストレージで保持） |
| F-20 | ブックマーク削除後に 5 秒間 Undo できる |
| F-21 | チェックボックスで複数選択し一括削除できる |
| F-24 | ドラッグ＆ドロップでブックマークの並び順を変更できる（並び順は DB に保存） |

## 画面一覧

| 画面名 | パス | 説明 |
|--------|------|------|
| ログイン画面 | `/sign-in` | Clerk が提供するログイン画面 |
| ブックマーク一覧 | `/bookmarks` | 自分のブックマーク一覧を表示。新規登録ボタンあり |
| ブックマーク新規登録 | `/bookmarks/new` | URL・タイトル・メモを入力して登録 |
| ブックマーク編集 | `/bookmarks/[id]/edit` | 既存ブックマークを編集 |
| タグ管理 | `/bookmarks/tags` | タグの作成・削除・一覧表示 |

## 非機能要件

| # | 要件 |
|---|------|
| NF-01 | TypeScript strict モードで実装する |
| NF-02 | Biome でコードフォーマット・リントを統一する |
| NF-03 | API ルート・Server Actions にはユニットテストを作成する |
| NF-04 | 開発環境は devcontainer で統一する |
| NF-05 | 本番 DB は PostgreSQL を使用する |
